# Example Spec

This is an example spec showing the format. Delete this file and write your own.

## Problem

Describe the problem or need this feature addresses. Why does it matter?

## Requirements

- Requirement one — what the system must do
- Requirement two — observable, testable outcome
- Requirement three — edge cases and constraints

## Edge Cases

- What happens when input is empty?
- What happens under load?
- What happens with invalid data?
